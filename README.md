# 🐱 月薪喵桌面宠物

治愈系桌面宠物 — 一只软萌的"月薪喵"风格小猫，陪伴你的每一天。
使用 PyQt5 + QMovie 实现 GIF 动画，支持 10 种状态切换。

## 🚀 使用方法

**开发环境：** 双击 `miaomiao_pet.bat`（需要 Python 3.12 + PyQt5）

**便携版（推荐分享）：** 解压 `月薪喵桌宠-Portable.zip`，双击 `月薪喵桌宠.bat` 即可，无需安装任何依赖。

## 📸 功能一览

| 功能 | 说明 |
|------|------|
| 🐾 透明桌宠 | 无边框透明窗口，始终置顶 |
| 🎬 GIF 动画 | QMovie 原生播放，每个状态随机选图 |
| 🖱️ 拖动移动 | 左键拖动，带屏幕边界限制 |
| 👆 点击互动 | 左键点击触发「喜欢」表情 + 语录，5秒恢复 |
| 🚶 自动走动 | 右键开始走动，自动巡逻，遇边转向 |
| 🏢 打工模式 | 右键开始打工，显示上班状态 |
| 🎵 MVP 结算 | 屏幕中央大图展示 + 背景音乐 |
| 💬 双语录系统 | 待机夸夸语录 + 打工主题语录 |
| ⚙️ 开机自启 | 右键一键开启/关闭 |
| 📐 调整大小 | 右键选择 100/150/200/250/300 |

```
miaomiao/
├── miaomiao_pet.bat         ← 本地启动脚本
├── main.py                  ← 主程序（PyQt5）
├── config.json              ← 配置文件（自动生成）
├── README.md
├── 月薪喵素材/               ← GIF 动画素材（按状态分文件夹）
│   ├── 放松/    (6张)       ← 待机默认状态
│   ├── 走动/    (8张)       ← 走动巡逻
│   ├── 上班/    (5张)       ← 打工模式
│   ├── 睡觉/    (3张)       ← 睡觉
│   ├── 在家/    (5张)       ← 在家休闲
│   ├── 摸鱼/    (3张)       ← 摸鱼状态
│   ├── 打招呼/  (1张)       ← 启动时打招呼
│   ├── 喜欢/    (13张)      ← 点击触发
│   ├── 生气/    (2张)       ← 生气表情
│   ├── 打工语录/ (6张)      ← 右键「今日打工语录」
│   └── MVP结算/ (1张)       ← MVP 结算弹窗
├── 音乐素材/                 ← 背景音乐
│   └── MVP结算/             ← MVP 结算音乐（.mp3）
└── MiaoMiaoPet-Portable/    ← 便携打包目录
```

## 🎨 10 种状态

| 状态 | 素材夹 | 说明 | 触发方式 |
|------|--------|------|---------|
| 放松 | 放松/ | 待机默认 | 启动 / 右键切换 |
| 走动 | 走动/ | 巡逻走动 | 右键「开始走动」 |
| 上班 | 上班/ | 打工模式 | 右键「开始打工」 |
| 睡觉 | 睡觉/ | 闭眼睡觉 | 右键切换 |
| 在家 | 在家/ | 休闲在家 | 右键切换 |
| 摸鱼 | 摸鱼/ | 摸鱼状态 | 右键切换 |
| 打招呼 | 打招呼/ | 启动问候 | 程序启动时（5秒自动恢复） |
| 喜欢 | 喜欢/ | 点击反应 | 左键点击猫猫（5秒自动恢复） |
| 生气 | 生气/ | 生气表情 | 右键切换 |
| MVP结算 | MVP结算/ | 结算弹窗 | 右键「MVP结算」 |

## 💬 语录系统

- **待机夸夸语录**：闲置时自动弹出，暖心鼓励铲屎官（20条）
- **打工主题语录**：右键「今日打工语录」触发，配合打工 GIF（11条）
- **点击语录**：左键点击猫猫时弹出（5条）
- 气泡显示 3-5 秒自动消失

## ⚙️ 配置 (config.json)

```json
{
    "pet_size": 150,
    "always_on_top": true,
    "start_position": null,
    "move_speed": 2,
    "enable_random_quote": true,
    "quote_interval_seconds": 30
}
```

## 🔧 技术实现

- **语言**：Python 3.12
- **GUI**：PyQt5
- **动画**：QMovie（原生 GIF 播放，逐状态随机选图）
- **音频**：QMediaPlayer（MP3 播放）
- **窗口**：无边框、透明背景、始终置顶
- **气泡**：半透明圆角 QLabel + 阴影
- **开机自启**：Windows 注册表 Run 键
- **边界限制**：拖动/走动均限制在屏幕可视区域内

## 💡 替换素材

在 `月薪喵素材/` 各子文件夹中放入 `.gif` 文件即可，程序会自动加载，切换状态时随机选取。
